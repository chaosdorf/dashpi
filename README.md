# [Chaosdorf Dashboard](https://wiki.chaosdorf.de/Dashboard) #

## Setup ##

```
sudo apt install ruby ruby-dev build-essential git bundler
git clone https://github.com/chaosdorf/dashpi.git
cd dashpi
bundle install
```

## Run ##

```
cd dashpi
smashing start
```
